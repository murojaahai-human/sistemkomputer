üìö Apa Itu Sistem Komputer?

Pengenalan Sistem Komputer

üí° Analogi Sederhana: Bayangkan komputer seperti tubuh manusia. Ada otak (CPU), memori (RAM), panca indera (input devices), dan cara berkomunikasi (output devices). Semua bekerja bersama untuk menjalankan tugas!

üéØ Definisi

Sistem komputer adalah kombinasi dari hardware (perangkat keras), software (perangkat lunak), dan brainware (pengguna) yang bekerja sama untuk memproses data menjadi informasi.

‚öôÔ∏è Fungsi Utama

Sistem komputer melakukan 3 proses utama:

  • Input - Menerima data
  • Process - Mengolah data
  • Output - Menghasilkan informasi

üåü Kenapa Penting?

Di era digital, sistem komputer ada di mana-mana! Dari smartphone yang kamu pakai, laptop untuk belajar, sampai sistem di bank dan rumah sakit. Memahami sistem komputer = memahami teknologi modern!

üìñ Glossary - Kamus Istilah

Klik untuk melihat definisi!

CPU (Central Processing Unit)
Otak dari komputer yang bertanggung jawab untuk memproses semua instruksi dari program. Semakin cepat CPU, semakin cepat komputer memproses data.
RAM (Random Access Memory)
Memori sementara yang digunakan untuk menyimpan data aplikasi yang sedang berjalan. Ketika komputer dimatikan, data di RAM akan hilang.
HDD (Hard Disk Drive)
Perangkat penyimpanan data permanen yang menggunakan piringan magnetik. Lebih lambat dari SSD tapi kapasitasnya besar dengan harga lebih murah.
SSD (Solid State Drive)
Perangkat penyimpanan modern yang menggunakan chip flash memory. Lebih cepat, tahan goncangan, dan hemat daya dibanding HDD.
GPU (Graphics Processing Unit)
Prosesor khusus untuk mengolah grafik dan visual. Penting untuk gaming, editing video, dan rendering 3D.
Motherboard
Papan sirkuit utama yang menghubungkan semua komponen hardware. Semua perangkat terhubung ke motherboard.
BIOS (Basic Input Output System)
Program yang tersimpan di motherboard untuk menginisialisasi hardware saat komputer dinyalakan sebelum loading sistem operasi.
Operating System (Sistem Operasi)
Software utama yang mengelola hardware dan menjalankan aplikasi. Contoh: Windows, macOS, Linux, Android, iOS.
Input Device
Perangkat yang digunakan untuk memasukkan data ke komputer. Contoh: keyboard, mouse, scanner, microphone, webcam.
Output Device
Perangkat yang menampilkan atau menghasilkan output dari komputer. Contoh: monitor, printer, speaker, proyektor.
Bandwidth
Kapasitas maksimal transfer data dalam jaringan. Semakin besar bandwidth, semakin cepat transfer data.
Driver
Software yang memungkinkan sistem operasi berkomunikasi dengan hardware. Setiap hardware membutuhkan driver yang sesuai.
Firmware
Software permanen yang tertanam di hardware. Contoh: BIOS di motherboard, software di printer.
Cache
Memori sangat cepat yang menyimpan data yang sering diakses untuk mempercepat proses. Ada di CPU, browser, dll.
Port
Konektor fisik untuk menghubungkan perangkat eksternal. Contoh: USB port, HDMI port, audio jack.
